# Caldera Forms Documentation Search

## Install
* `git clone git@github.com:CalderaWP/caldera-forms-docs-search.git`
* `cd caldera-forms-docs-search npm install`

# Development
* `npm run start`

# Deploy
* `npm run deploy`
